PROFESSIONAL PATH

Nama saya Leska Naturalisa, Saya lulusan Universitas Gunadarma dengan jurusan Teknik Informatika Tahun 2019 dan saya angkatan 2015. Selama saya kuliah banyak sekali hal pengalaman dan ilmu yang saya dapatkan baik di dalam lingkungan kampus maupun di luar kampus. Pengalaman pertama saya kuliah diawali dengan mengikuti organisasi kampus yaitu Himpunan Teknik Informatika serta saya menjadi Asisten Laboratorium Teknik Informatika Universitas Gunadarma pada saat saya semester 2. Kegiatan saya menjadi asisten Lab yaitu membimbing mahasiwa ketika belajar tentang mata kuliah yang mengharuskan praktikum, saya memberikan pengajaran mengenai dasar-dasar bahasa pemrograman Java, Python, Ruby, Visual Basic, dan Delphi. Banyak sekali ilmu yang saya dapatkan dalam Lab. Kemudian masuk semester 7, saya menjadi staff sekertariat dosen Universitas Gunadarma. Lalu kegiatan di luar kampus, saya mengikuti bootcamp yang diadakan oleh Keboen Koding dan mempelajari JavaScript, mengikuti beberapa seminar tentang industri 4.0, android development, serta beberapa workshop. Memasuki perkuliahan tingkat akhir, Kominfo bekerja sama dengan Universitas saya untuk mengadakan Digital Talent Scholarship 2019 dan saya menjadi asisten pengajar selama kegiatan itu berlangsung. Selama saya berapa semester menengah perkuliahan, saya mulai tertarik dengan web development. Menurut saya, Web sangat menantang dan banyak sekali materi tentang website yang sangat asik untuk dipelajari.
